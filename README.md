Data Driven methods to Predict Turbulent/ Eddy Viscosity

In this repository we will implemnt SINDy algorithm a sparse identification for non  linear dynamics for a fluid flow problem flow past square cylinder.
The mentioned case was run in OpenFOAM and data postprocessng and data collection was done in Paraview. We are coparing the predicted turbulent / eddy viscosity with standard turbulence models like k-w SST model, k-epsilon model, etc.
